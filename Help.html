This is an User Interface (ui) application which implements in-vitro dissolution profile comparison.

Please visit the page, https://github.com/jszlek/BEMoDA_shiny for more information on the application.

In general the BEMoDA_shiny utilizes two approaches proposed by Tsong et al. in 1996 - model independent approach and by Sathe et al. in 1996 - model dependent approach.

Model independent approach


According to Tsong et al. model independent approach can be utilized to dissolution profile comparison when:

1. Similarity limit of the pre-change product is well-defined (Dg = mean % of difference)

2. Dissolution data of the reference product cannot vary significantly from batch to batch.

3. Dissolution testing of the test and the reference batches are made under identical conditions and with identical number of units.

4. Dissolution profiles have to be adequately represented.

5. Dissolution values are sampled at point that adequately represents dissolution profile.


Under this assumptions to compare dissolution profiles of test and reference batches in model independent manner the user need to:


1. Prepare data sets for test and reference.

2. Load data sets.

In the main window click on the “Reference” tab. Set the proper data format and click “Browse” to navigate to your reference dissolution data. If the data is correctly loaded the user should see the “spaghetti plot” of dissolution data.



To load the test data click on the “Test” tab.

Please be advised, in model independent approach as stated above the user do not need the dissolution data for Standard batches to establish inter- and batch-to-batch variability.


3. Modify settings according to user needs.




Click on the tab “Profile comparison” and select “Model independent” option. Set the mean percentage of difference for batch-to-batch and inter-lot units. Vector of probabilities is density distribution, quantile function for the F distribution with degrees of freedom df1 = no.of.time.points, df2 = no.of.units.ref + no.of.units.test – no.of.time.points - 1


4. Run calculations.


Click on “Run calculations” button.


Model dependent approach


In Model dependent approach the user need to provide more dissolution data to establish inter- and intra-lot variance for reference product. Therefore it is advisable to prepare data for 3 standard batches of 12 units each. According to Sathe et al. the following steps need to be considered:

1. Definition of mechanistic model suitable to describe dissolution data of standardized production size lots or batches.

2. Asses the inter- and intra-batch variability through fitting the individual unit dissolution data.

3. Define similarity region (SR) based on the variances of standardized batches parameters.

4. Using model chosen in step 1, asses the model parameters of the reference (pre-change) and test (post-change) batches by fitting the dissolution data.

5. Calculate statistical distance between the mean parameters.

6. Define 90% confidence region (CR) around statistical distance.

7. Compare CR and SR to declare similarity or dissimilarity.


In order to perform model dependent profile comparison:


1. Prepare data sets for 3 standard batches of reference product (preferably N = 12 each batch).


2. Prepare data sets for test (post-change) and reference (pre-change) product. The structure of data sets need to be in previously mentioned format.


3. Load data sets.

In the main window click on the “Reference” tab. Set the proper data format and click “Browse” to navigate to your reference dissolution data. If the data is correctly loaded the user should see the “spaghetti plot” of dissolution data.

To load the test data click on the “Test” tab and perform the same operations like on previous tab. Click on “Standard” tab to load dissolution data for 3 standard batches.


4. Modify settings according to user needs.


Click on the tab “Profile comparison” and select “Model dependent” option.

From the left-hand side options available:

1. At the first column the user can

- select mechanistic model

- chose method for optimization of model parameters (+ no of maximum iterations and relative tolerance)

2. The second column is devoted to settings of similarity (SR) and critical region (CR) parameters as:

- SR confidence interval

- CR confidence interval

- no of points detected at boundary of SR and CR

- draw SR as rectangle or ellipse (when rectangle is selected no of points detected at boundary of SR has no effect)

- log-normal transformation of CR and SR

3. At the third column the user can chose maximum no of steps for optimization of coordinates of SR and CR together with their relative tolerance.


Model dependent approach implemented in BEMoDA_shiny application is focused on the steps 2 – 7. Therefore it is recommended to use more suitable software to chose the best model to fit the dissolution data.
